Lazily Initialized
==================

a blog about life, learning, and laziness


## Development
1. `git clone` this repo
2. `npm install && bower install`
3. `grunt serve`

#### Compiles: 
- CoffeeScript (`scripts/*.coffee`)
- Handlebars (`templates/*.hbs`)
- Sass (`styles/*.s{a,c}ss`)
- Markdown (`posts/*.md`)
